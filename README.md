# 介绍 - About
中式工坊：新生,旨在打造一个相对精简的中式风格建筑模组，是对原本中式工坊的重置，对纹理进行重绘以及添加一些方块和特性。
也考虑加入更多近代中式风格元素。
Chinese Workshop in higher versions and more loader, add some new features.
